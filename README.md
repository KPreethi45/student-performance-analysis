# 🎓 Student Performance Analysis using Python

## 📌 Project Overview

This project focuses on analyzing student performance data to uncover meaningful insights and trends. The dataset consists of 1000 students with details such as gender, parental education, test preparation, and subject scores.

The goal of this project is to apply data analysis techniques to understand factors affecting student performance and visualize key insights.

---

## 🚀 Key Features

* 🔹 Data Cleaning and Preprocessing
* 🔹 Feature Engineering (Total & Average Score Calculation)
* 🔹 Topper Identification (Highest Scoring Student)
* 🔹 Analysis of Test Preparation Impact on Performance
* 🔹 Grade Classification System (A–F)
* 🔹 Data Visualization using Matplotlib and Seaborn

---

## 📊 Dataset Details

* Total Records: **1000 students**
* Features:

  * Gender
  * Race/Ethnicity
  * Parental Level of Education
  * Lunch Type
  * Test Preparation Course
  * Math Score
  * Reading Score
  * Writing Score

---

## 🛠️ Technologies Used

* Python 🐍
* Pandas 📊
* Matplotlib 📉
* Seaborn 🎨

---

## 📈 Key Insights

* 📌 Students who completed the test preparation course tend to score higher
* 📌 Balanced performance observed across reading and writing compared to math
* 📌 Top-performing student achieved a perfect score (300/300)
* 📌 Clear variation in performance based on different factors
